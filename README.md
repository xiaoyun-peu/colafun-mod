# colafun-mod
Play Minecraft? Get some Cola!

#这个仓库仅用于报告问题！
#This repository is only for reporting issues!

## 中文（Chinese）:<br>
是时候为Minecraft增加一些汽水了！<br>
**前期提要**！这个模组含有**糟糕的美术成分**！但兼容性较好。<br>
本模组新增加了一种“矿石”——石灰石<br>
你可以使用他们来烧炼二氧化碳并与糖、棕色染料、水瓶、一个空可乐瓶（或空易拉罐）来制作汽水。<br>
本模组所有内容可以合成，具体请参阅Wiki<br>
<br>
## English(英文):<br>
It's time to add some cola to Minecraft!<br>
**Early heads-up**! This mod contains **poor art assets**! But it has good compatibility.<br>
This mod adds a new "ore" — limestone.<br>
You can use it to smelt carbon dioxide and combine it with sugar, brown dye, a water bottle, an empty soda bottle (or an empty can) to make soda.<br>
Everything in this mod can be crafted; please refer to the Wiki for details.<br>
